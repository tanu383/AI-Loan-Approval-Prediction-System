# AI Loan Approval Prediction System

An AI-powered financial prediction system built using Machine Learning and Streamlit.

## Features

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Multiple ML Models Comparison
* Random Forest Classifier
* Interactive Streamlit Dashboard
* Financial UI Design

## Models Used

* Logistic Regression
* KNN
* Naive Bayes
* Random Forest

Random Forest gave the best performance among all models.

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* Matplotlib
* Seaborn
